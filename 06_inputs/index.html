Part 1: Capacitive Sensor, Petal Pressed Signaler

I wanted to create a capacitive sensor that when pressed could turn on an LED, signalling that there was a change in pressure. This could be used for a step petal, if a petal is pressed a light flickers off, creating a visual signal that the petal has been pressed.


Here, you can see that the sensor is printing an updated description of the change in voltage in arbritrary numbers. We can see that when the sensor is touched, the arbritrary updated voltage becomes very small and usually negative. Then, when the voltage drop is large, the LED will turn off temporarily.
